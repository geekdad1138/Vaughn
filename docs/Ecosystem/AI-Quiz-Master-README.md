# AI Quiz Master

An open-source educational platform delivering comprehensive AI quizzes across multiple formats and delivery mechanisms.

## Vision

Make AI education accessible to everyone through:
- **Printable PDFs** for classroom and offline use
- **Interactive web platform** for online learning
- **CLI tool** for programmers and developers
- **Comprehensive content** from K-12 to advanced professionals
- **Free and open-source** for the community

## Project Structure

```
AI-Quiz-Master/
├── quizzes/                 # Quiz content (markdown format)
│   ├── school/             # K-12 grade-level quizzes
│   ├── professional/       # Topic and skill-based quizzes
│   └── master-class/       # Comprehensive mastery quiz
├── docs/                    # Documentation
│   └── spec/               # Project specifications
├── README.md               # This file
├── CONTRIBUTING.md         # Contribution guidelines
└── LICENSE                 # MIT License
```

## Quiz Categories

### School Level (K-12)
- Introductory AI (K-5)
- Intermediate AI Understanding (6-8)
- Advanced AI Concepts (9-12)

### Professional Level
- **Topics**: Natural Language Processing, Prompt Engineering, Voice Processing, LangChain, LLMs, and more
- **Expertise Levels**: Beginner, Intermediate, Advanced

### Master Class
- Comprehensive quiz covering all major AI topics
- 100+ questions with weighted scoring
- Reportable certification
